  • Nano Hacker Squad

    URL: http://nanohackersquad.blogspot.com/

    I helped setup a small very young community of coders, known as the nano hacker squad. It was an exploration on teaching tech for kids but also about collaborative learning and making. The goal was to introduce them to the hackathon culture. This is by far my favorite project of 2014. I was the organizer and lead mentor coming up with coding activities for the kids. We had few other dedicated mentors helping out: Benno, Ricky, Matt and few others.

    Article by Thoughtworks Video by Thoughtworks

    One of our young participants was Kiera who developed a couple of ios apps and received an Apple scholarship. We did not teach ios. Kiera learnt ios by herself! I used my skills on After Effects to help with her video submission.

    It's a lovely small engaged talented community still going strong in 2015.

  • Green Screening with WebRTC

    http://greenscreen.surge.sh/

    This is an exploration of web standards for HTML5 <video>, <canvas> and webrtc that enables real-time video processing, in particular on how to accomplish green screening effects (for free!)

    It was presented at the New York Times Open Science Fair 3.0 in July, 2015.

    My original motivation was to hack existing online cool effects based on HTML5 <video> by hijacking the <video> tag and changing the video source to the web camera using webrtc's Media Capture API.

    Then when Seriously.js was out with the green screening, I couldn't resist trying it out with the hack.

    I presented it at the Hack n Tell meetup, then I was approached to present it at the NYT Open Source Science Fair.

    This is a lot of fun to present to technologists. It's like magic.

  • Typographics with SVG Fonts

    URL: ATCG

    Instructions: Press n/N, i/I, r/R to change Noise, Intensity, Richness

    This is an interactive HTML5 animation of breaking letters into noisy shapes.

    Originally, I twiddled with some Processing code for some other prototype based on: http://www.generative-gestaltung.de/P_3_2_1_01

    To get my desired effect of breaking into noisy shapes using Processing. FontLineDots

    I wanted to reproduce the same interactive breaking effect using HTML5 but there's no equivalent font processing library on the Javascript side. However, I worked extensively with parsing SVG fonts using JS in some previous exploration. And I explored how to break the SVG into multiple lines with the PolyLineSegmentorSvgCanvasExecutor.

    I'm very pleased with the result since it can be used with most SVG graphics (not just letters) but I'm quite devoted to typographic effects.

  • Without Title by Lia

    URL: Without Title by Lia

    This is a Javascript port of Lia’s Without Title. The original can be seen in her Java applet. You should check some of Lia’s really amazing projects.

    Although this is subjective, the visual quality didn’t equal to the original. It’s my impression that the smooth() function is not working as expected.

    The drawing performance in the Javascript version is notorious. It feels very responsive. Personally, I find the slower Java/Processing version more pleasant and relaxing. I don’t like to slow things down but it may improve the experience in this case.

    You can click/move the mouse to interact with the drawing and press any key to pause.

  • Rose Fabric
